  ## Usage

  Place `AuroraBackground` inside a relative container. It renders blurred gradient blobs that float with CSS animations. Perfect for hero sections and landing pages.

  ### Basic

  <ExampleBlock
    codeDefaultOpen
    code={`import { AuroraBackground } from "@glinui/ui"

export function AuroraBackgroundDemo() {
  return (
    <div className="relative flex h-64 w-full items-center justify-center overflow-hidden rounded-xl border border-neutral-200 bg-neutral-950 dark:border-neutral-800">
      <AuroraBackground />
      <span className="relative z-10 text-3xl font-bold text-white">Aurora</span>
    </div>
  )
}`}
  >
    <div className="relative flex h-64 w-full items-center justify-center overflow-hidden rounded-xl border border-neutral-200 bg-neutral-950 dark:border-neutral-800">
      <AuroraBackground />
      <span className="relative z-10 text-3xl font-bold text-white">Aurora</span>
    </div>
  </ExampleBlock>

  ## Reduced Motion

  When `prefers-reduced-motion: reduce` is active, all blob animations are paused via `motion-reduce:[animation:none]`. The static gradient blobs remain visible.
