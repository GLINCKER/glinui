  ## Usage

  Wrap text in `ChromaticText` to apply RGB channel splitting. Pass a string as `children` — the component renders ghost layers with offset colors.

  ### Basic

  <ExampleBlock
    codeDefaultOpen
    code={`import { ChromaticText } from "@glinui/ui"

export function ChromaticTextDemo() {
  return (
    <div className="flex items-center justify-center p-8 bg-neutral-950 rounded-xl">
      <h2 className="text-4xl font-bold text-white">
        <ChromaticText>Glin UI</ChromaticText>
      </h2>
    </div>
  )
}`}
  >
    <div className="flex items-center justify-center p-8 bg-neutral-950 rounded-xl">
      <h2 className="text-4xl font-bold text-white">
        <ChromaticText>Glin UI</ChromaticText>
      </h2>
    </div>
  </ExampleBlock>

  ## Accessibility

  - Ghost layers are `aria-hidden="true"` — screen readers only see the base text.
  - The base text remains fully readable and accessible.

  ## Reduced Motion

  When `prefers-reduced-motion: reduce` is active, the animated chromatic shift is paused via `motion-reduce:[animation:none]`. The static offset layers remain visible.
