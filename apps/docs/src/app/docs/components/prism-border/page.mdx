  ## Usage

  Wrap any content with `PrismBorder` to add an animated rainbow refraction border. Uses CSS mask compositing for the hollow border effect.

  ### Basic

  <ExampleBlock
    codeDefaultOpen
    code={`import { PrismBorder } from "@glinui/ui"

export function PrismBorderDemo() {
  return (
    <div className="flex items-center justify-center p-8">

        <div className="rounded-xl bg-neutral-900 p-6 text-center">
          <h3 className="text-lg font-semibold text-white">Prism Card</h3>
          <p className="mt-2 text-sm text-neutral-400">Rainbow border effect.</p>
        </div>
      </PrismBorder>
    </div>
  )
}`}

        </div>
      </PrismBorder>
    </div>
  </ExampleBlock>
