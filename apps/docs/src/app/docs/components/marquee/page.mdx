  ## Usage

  Place any elements as children. The marquee duplicates them to create a seamless infinite scroll loop.

  ### Basic

  <ExampleBlock
    codeDefaultOpen
    code={`import { Marquee, Badge } from "@glinui/ui"

export function MarqueeDemo() {
  return (
    <Marquee className="py-4" gap={24}>
      {["React", "Next.js", "Tailwind", "Radix", "TypeScript", "Vitest"].map((t) => (
        <Badge key={t} variant="glass" className="mx-2 text-sm">{t}</Badge>
      ))}
    </Marquee>
  )
}`}
  >
    <Marquee className="py-4" gap={24}>
      {["React", "Next.js", "Tailwind", "Radix", "TypeScript", "Vitest"].map((t) => (
        <Badge key={t} variant="glass" className="mx-2 text-sm">{t}</Badge>
      ))}
    </Marquee>
  </ExampleBlock>

  ## Accessibility

  - The duplicated children are marked `aria-hidden` to prevent screen readers from reading content twice.
  - Avoid placing interactive elements inside the marquee.

  ## Reduced Motion

  Animation uses `motion-safe:animate-marquee-x/y`. Under `prefers-reduced-motion`, the content stays static.
