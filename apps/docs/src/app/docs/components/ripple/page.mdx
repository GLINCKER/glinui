  ## Usage

  Place `Ripple` inside a relative container. The rings expand outward from the center.

  ### Basic

  <ExampleBlock
    codeDefaultOpen
    code={`import { Ripple } from "@glinui/ui"

export function RippleDemo() {
  return (
    <div className="relative flex h-48 w-full items-center justify-center overflow-hidden rounded-xl border border-neutral-200 bg-white dark:border-neutral-800 dark:bg-neutral-950">
      <span className="z-10 text-sm font-medium">Ripple</span>
      <Ripple />
    </div>
  )
}`}

      <span className="z-10 text-sm font-medium">Ripple</span>

  ## Accessibility

  - Ripple is purely decorative with `pointer-events-none`.
  - No interactive or focusable elements.

  ## Reduced Motion

  Ring animations use `motion-safe:animate-[...]` / `motion-reduce:hidden`. Under `prefers-reduced-motion`, rings are hidden entirely.
